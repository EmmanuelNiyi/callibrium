# callibrium
⚕️ Callibrium: A constraint-based scheduling system for creating fair and compliant medical intern call rosters. Built with Python and Google OR-Tools CP-SAT solver.

**Key Features**:
- 🏥 Automatic generation of duty schedules with medical constraints
- ⚖️ Fair shift distribution across interns
- 🚫 Configurable constraints (max shifts, no consecutives, blackout dates)
- 📊 Visual schedule export (HTML/PDF)
- 🧠 Smart conflict resolution engine

**Use Cases**:
- Teaching hospitals
- Residency programs
- Medical internship coordination
- Emergency department staffing

**Tech Stack**:
- Python 3.10+
- OR-Tools Constraint Programming
- FastAPI (Web Interface)
- Pandas (Data Handling)
- Plotly (Visualization)
